# Finance Budget vs Actual Analysis (finance projectttt)

This project analyzes company budgeted vs actual spending using Python and Power BI.

## Tools Used
- Python (Pandas)
- Excel
- Power BI

## Workflow
1. Raw data collected in Excel
2. Data cleaned using Python
3. Clean data exported to CSV
4. Interactive dashboard built in Power BI

## Key Insights
- Total budget vs actual spending
- Department-wise comparison
- Monthly spending trends
- Variance analysis

## Files
- Raw Excel file
- Cleaned CSV file
- Jupyter Notebook
- Power BI dashboard
